# Rafael Correa - Obras y Trayectoria

Sitio web sobre Rafael Correa: obras, trayectoria, opiniones y datos visuales.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Build

```bash
npm run build
npm start
```

## Estructura

- `app/` - Páginas y rutas
- `components/` - Componentes React
- `data/` - Datos JSON (obras, opiniones, timeline)

Para más detalles, ver `LEER-PRIMERO.txt`.
